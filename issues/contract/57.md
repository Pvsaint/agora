# Issue 57: Create Referral rewards mechanism

### DESCRIPTION

> **Incentivize growth through community participation.** Referral marketing is powerful; this issue implements an on-chain referral system where referrers are instantly rewarded a percentage of the platform fee for every seat they fill.

### TASKS

- [ ] **Add a `referrer: Address`** argument to the purchase function.
- [ ] **Calculate the referral reward** (e.g., 20% of the `platform_fee`).
- [ ] **Execute the reward distribution** from the escrowed funds to the referrer address upon successful payment.
- [ ] **Add a 'No Self-Referral' check** to prevent buyers from referring themselves to get a discount.
